Game Instructions
  1. There are 60 seconds per game
  2. A names of the countries pops up randomly so you type it in the input box, as many as you can
  3. You get 10 points per correct typing
  4. At the end of each game, you receive a medal depending on the score
    • More than 600 points - Gold Medal
    • Between 310 and 590 points - Silver Medal
    • Less than 300 points - Bronze Medal
  5. To start the game (& restart the game), type "Ireland" in the input box
