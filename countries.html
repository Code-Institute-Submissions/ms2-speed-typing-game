Game Instructions
  1. There are 60 seconds per game
  2. A name of the countries pops up randomly so you type it in the input box
  3. You get 10 points per correct typing
  4. At the end of each game, you receive a medal depending on the score
    • Gold Medal - More than 600 points
    • Silver Medal - Between 310 and 590 points
    • Bronze Medal - Less than 300 points
  5. To start (& restart) the game, click Start button below input box

*** IMPORTANT ***
Speed Typing game is available only on Laptop / Desktop as a key board is required to play it
